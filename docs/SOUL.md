# Project Soul

## Vision
To eliminate the cognitive friction of managing software on Windows by providing the ultimate "pacman-like" experience.

## Principles
1. **Unified Simplicity:** One command to rule all package managers.
2. **Native Performance:** No heavy runtimes; leverage the power of PowerShell's native concurrency.
3. **Information Density:** Clear, normalized, and actionable output.
4. **Portability:** A single-module solution that works across environments with zero dependencies.
5. **Transparency:** Decrypting cryptic Windows error codes into human-readable diagnostics.

## Values
- **Speed is a Feature:** Searches should be concurrent and instant.
- **Reliability over Magic:** Explicit logic and robust parsing over unstable heuristics.
- **User Empowerment:** Providing the tools to "doctor" your own system health.
